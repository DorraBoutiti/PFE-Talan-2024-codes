Important informations to extraction from an employment contract:
    - Job details
        - Job title
        - Responsabilities
        - remote / office
        - work schedule
        - Working hours
        - vocation days
        - sick days
    - Compensations & benifits 
        - salary or hourly rate
        - bonuses / commissions
        - health insurance
        - paid time off
    - Termination clause
    - confdentiality 
    - Dispute resolution
    - Employer name
    - Employe name
    - date of signing the contract
    - Start date
